# Race Data Karaoke

Transform motorsport data into entertaining party experiences for true racing fans!

## Overview

Race Data Karaoke is an interactive platform that turns racing data into stunning visualizations paired with AI-generated commentary. Pick a visualization type, choose a commentary style, and perform like a real race announcer.

## Getting Started

Access the app on Poe at https://poe.com/Race_Data_Karaoke.  Login with Google or other option to activate a free account - no credit card required.  The free account provides 3000 tokens per day. These are the tokens used to run the app's AI features like generating the visuals and commentary.

## How It Works

1. Select a visualization type that interests you
2. Choose your preferred commentary style
3. The AI generates custom commentary tailored to your selections
4. Watch the visualization animate while you perform the commentary out loud

## Demo Data

The app comes with real racing data from the 2024 Toyota GR Cup at Barber Motorsports Park, so you can start exploring immediately.
